# 仓库管理系统 java358

## 项目概述

仓库管理系统是一个基于前后端分离架构的线上物品平台，主要用于仓库信息管理。系统提供丰富的功能模块，通过后台管理体系，实现对仓库物品的高效管理。

## 技术栈

- 后端：Java、SpringBoot
- 前端：Vue
- 数据库：Mysql

## 系统角色

- 用户：可进行物品查询、操作日志查看和个人信息管理。
- 管理员：除用户功能外，可管理物品信息、用户信息和操作日志。

## 运行环境

- 系统环境：Windows、Mac
- 开发环境：IDEA、JDK 1.8、Mysql

## 功能模块

### 用户功能

- 首页：显示当前仓库时间和使用人。
- 物品管理：查询仓库物品及状态。
- 操作日志：查看仓库操作分类、操作人、申请人及时间。
- 个人中心：管理个人信息。

### 管理员功能

- 首页：显示当前仓库时间和使用人。
- 物品信息管理：重置、查询、编辑仓库信息。
- 物品分类管理：重置、查询、编辑物品信息。
- 物品管理: 新增、入库、出库、重置、查询、编辑物品信息。
- 操作日志：查看仓库操作分类、操作人、申请人及时间。
- 个人中心：管理个人信息。
- 用户信息管理：修改、编辑、删除用户信息，以及新增用户。
